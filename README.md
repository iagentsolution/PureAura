# ✨ PureAura | by @iagentsolution

Purificá el caos, dominá tu ritmo. 

PureAura no es solo un asistente; es un motor de análisis conductual diseñado para decodificar tu bienestar, productividad y ritmo vital. A través de una interfaz minimalista en Streamlit, PureAura analiza tu lenguaje y autopercepción para ofrecerte un diagnóstico clínico en tiempo real.

## 🚀 Características Únicas
- Dashboard de Aura: Visualización dinámica de Energía, Estrés y Foco basada en análisis de texto.
- Contexto Astrológico y Estacional: Sincronización con tu signo zodiacal y la fase climática actual (Hemisferio Sur).
- Arquetipos de Usuario: Personalización de la experiencia mediante perfiles (Ninja, Astronauta, Robot, etc.).
- Reportes Profesionales: Generación automática de diagnósticos en formato PDF.

## 🛠️ Tecnología
- Lenguaje: Python
- Frontend: Streamlit
- Cerebro: Llama 3.3 (vía Groq Cloud)
- Documentación: FPDF2

## 📦 Instalación y Uso
1. Clona este repositorio.
2. Instala las dependencias: pip install -r requirements.txt
3. Ejecuta la app: streamlit run app.py (sustituye app.py por el nombre de tu archivo).
